# 💬 Amazon Reviews — Sentiment Analysis
### CodeAlpha Data Analytics Internship | Task 4

## 📌 Project Overview
Sentiment Analysis on Amazon Product Reviews using Python and NLP. Reviews classified as Positive, Negative, or Neutral based on polarity score.

## 🛠️ Tools & Libraries
- Python, Pandas, TextBlob, WordCloud, Matplotlib, Seaborn

## 📁 Dataset
- Amazon Product Reviews (30 reviews)
- Features: Review text, Star Rating, Sentiment, Polarity, Subjectivity

## 🔍 Analysis Steps
1. Data Loading & Creation
2. Sentiment Scoring using TextBlob
3. Polarity & Subjectivity Analysis
4. Word Cloud Generation
5. Visualization & Key Insights
